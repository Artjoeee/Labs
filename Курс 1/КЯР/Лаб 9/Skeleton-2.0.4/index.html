Класс grid

Сетка

Сетка (grid)представляет собой совокупность пересекающихся горизонтальных и вертикальных линий, разделяющих пространство gridконтейнера на области сетки, в которые могут быть помещены содержимое элементов сетки. Контейнер ведет себя как блочный элемент, заполняя 100 % доступной ширины.
Для определения области сетки для одной ячейки установить свойства grid-column-start(grid-column-end) и grid-row-start(gridrow-end), значениями которых указывается номер линии сетки. Можно использовать сокращенную запись grid-row: n/m; (gridcolumn: n/m;), где n является номером линии, с которой начинается область сетки.
Предварительно необходимо создать grid-контейнеры с помощью display: grid или display: inline-grid. Элемент с классом grid становится контейнером сетки, его потомки элементами сетки и занимает 100% доступной ширины. Свойства grid-template-columns и gridtemplate-rows устанавливают размер колонок и строк сетки.
Есть компоненты сетки, к которым относятся ячейка, область сетки, полоса сетки. Полосой сетки является расстояние между двумя соседними линиями сетки. Ячейка сетки — пространство в сетке, где пересекаются горизонтальная и вертикальная полосы сетки. Одна или несколько ячеек составляет область сетки.
